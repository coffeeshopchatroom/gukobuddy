# **App Name**: UniMate

## Core Features:

- Dynamic Flashcard Generation & Study: Create custom flashcard sets by manually inputting questions and answers or importing from text files. Review cards using a dedicated study mode.
- Organized Task Management: A comprehensive task list feature to add, prioritize, and track academic assignments, deadlines, and personal to-dos. Supports categorization by course or project.
- Digital Notebooks: A simple rich text editor for capturing and organizing notes by subject or topic. Allows for easy searching and retrieval of important information.
- Academic Progress Tracker: Input and monitor assignment scores, exam results, and course weightings. Automatically calculates current grades and estimated GPA for individual courses and overall.

## Style Guidelines:

- Primary color: A calming sage green (#A5D6A7) for main UI elements and a sense of tranquility.
- Background color: Clean white (#FFFFFF) providing a spacious and uncluttered canvas.
- Accent color: A soft light pink (#FFCDD2) used to highlight interactive elements and important call-to-actions.
- Headlines font: 'Cal Sans' for a distinctive and modern feel, suitable for key titles and section headings.
- Body font: 'IBM Plex Sans Devanagari' for its excellent readability and neutral, professional aesthetic across all text sizes, ideal for notes and task descriptions.
- Use a consistent set of clean, modern, and intuitive icons that visually represent each tool and function, ensuring quick user recognition.
- Implement a dashboard-like home screen allowing users quick access to all core tools. Each tool will have a structured, uncluttered layout emphasizing content and functionality.
- Incorporate smooth and slow animations for transitions and interactive elements, prioritizing aesthetics and a fluid user experience over immediate feedback.
